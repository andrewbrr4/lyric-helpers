# lyric-helpers

A repo for various jupyter notebooks, functions and classes with the common theme of using NLP/AI technologies to assist with song lyric writing.

Uses the conda environments `lyrics-helpers`.

## Notebooks

**word_associations.ipynb**
Finds words that are semantically similar--but not *too* similar--to the target word. The purpose is not just to serve as a simple thesaurus for substituting specific words in a line (thought that is a potential use case), but for more broad lyrical brainstorming and for building word-banks around a particular concept.
